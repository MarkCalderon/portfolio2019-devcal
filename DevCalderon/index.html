UPEVEN | eCommerce Site

It is an online transaction site where users are allowed to buy and sell over the internet. It includes a content management system where users are able to do basic functions of CRUD (Create, Retrieve, Update and Delete). It also comes with a security roles where administrators, sellers and buyers have different access from each other. The system was built with an SQL database where data is stored. This is our final year project for my major in BSc (Honours) in Computer Science and Software Engineering.

Languages:

C# ASP.NET MVC, HTML, CSS, jQuery and Microsoft SQL

Project Specifications:

Software Project Management includes the flowchart of the entire system which showcases the main functions of the site, development timeline and software architecture.

The system is built around the Model View Controller framework which separates the business model, user interface and the functionality.

At the business model, each data field is normalized to eliminate any redundancy of data.
